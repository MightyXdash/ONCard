# ONCard v1.4.7

![ONCard 1.4.7 update banner](https://raw.githubusercontent.com/MightyXdash/ONCard/main/assets/updates/1.4.7/whats_new_top_banner_16x9.png)

ONCard 1.4.7, "More Than One Choice...", is a minor update focused on MCQ study support, NNA uniformity, and stronger generation pipelines.

## Added

- Added MCQ study mode for easier, more interactive practice.
- Added the interactive recipe app UI.

## Improved

- Stabilized settings dialog motion state and slider interactions.
- Fixed the settings header icon artifact and increased navigation icon size.
- Redesigned settings navigation with icon-based sections and a modal shell.
- Further enhanced NNA uniformity so it can serve the whole app more consistently.
- Customized FTC and image processing pipelines to help generation for FTC and Ask AI.
- Cleaned up README structure and installation guidance.

## Notes

- Build and release artifacts for `1.4.7` should use the `ONCard-Setup-1.4.7.exe` naming convention.
- Replace the `1.4.7` update banner assets before publishing the GitHub release so release note image URLs render correctly.
- MCQ grading is still in beta. Marks will be either 10 or 0 while a better grading system is being developed.
