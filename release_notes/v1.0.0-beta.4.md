# ONCards v1.0.0-beta.4

Rebuilt the Windows release pipeline with PyInstaller.

![ONCards](https://github.com/user-attachments/assets/cd88364e-904a-41d1-87ff-0444d2cd70a7)

- Switched Windows packaging from Nuitka to PyInstaller.
- Rebuilt the installer so bundled icons and banners ship from the PyInstaller app layout.
- Kept the in-app update checker and GitHub release notes support.
- The installer only installs ONCards itself. Ollama/model downloads still happen inside the app wizard after install.
