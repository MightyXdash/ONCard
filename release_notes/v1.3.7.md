# ONCard v1.3.7

![ONCard 1.3.7 update banner](https://raw.githubusercontent.com/MightyXdash/ONCard/main/assets/updates/1.3.7/whats_new_top_banner_16x9.png)

ONCard 1.3.7, "A Better Yourself", focuses on helping learners know themselves better, with stronger Ask AI tools, cloud model support, multi-account workflows, and clearer performance feedback.

## Added

- Added a stats tool for viewing performance across selected time periods.
- Added card browsing for Ask AI.
- Added vision response support for Ask AI.
- Added selectable Ask AI tones.
- Added selectable emoji usage for Ask AI.
- Added support for more AI models.
- Added account export, import, delete, and swap support.
- Added support for multiple accounts, with stable use around 6 to 10 accounts.
- Added Ollama Cloud support.
- Added support for additional Ollama text models.
- Added newer and better system prompts for AI models.
- Added custom windows and blur effects beyond the default window style.
- Added version-specific update artwork for the 1.3.7 "Know Yourself" flow.

## Improved

- Storage and RAM usage are more efficient.
- The UI is more consistent, with more work still planned.
- Ask AI context length increased from 8k to 9k.
- Text input and output streaming is more optimized.
- Animations were updated.
- Cloud mode is off by default and overrides local text-model selection only while enabled.
- Embedding remains local even when cloud inference is enabled.

## Fixed

- Fixed an issue where Settings -> New Profile -> Add profile could not add existing accounts.
- Fixed account loading problems.
- Fixed account export security so API keys are not embedded across exported accounts.
- Fixed loading screen bugs and lazy loading for certain files.
- Fixed markdown problems for native function-calling models.
- Fixed streaming bugs in certain widgets and AI features.

## Notes

- Build and release artifacts for `1.3.7` should use the `ONCard-Setup-1.3.7.exe` naming convention.
