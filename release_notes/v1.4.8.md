# ONCard v1.4.8

![ONCard 1.4.8 update banner](https://raw.githubusercontent.com/MightyXdash/ONCard/main/assets/updates/1.4.8/whats_new_top_banner_16x9.png)

ONCard 1.4.8 is a patch release for the MCQ update.

## Fixed

- Fixed Nomic embedding model detection when Ollama reports the installed model as `nomic-embed-text-v2-moe:latest`.
- The app now accepts both `nomic-embed-text-v2-moe` and `nomic-embed-text-v2-moe:latest` while checking NNA/semantic model availability.

## Included

- MCQ study support and the 1.4.x update artwork remain included.
- Release artifacts for `1.4.8` use the `ONCard-Setup-1.4.8.exe` naming convention.
