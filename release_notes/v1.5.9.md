# ONCard v1.5.9

ONCard 1.5.9 is the real next version after 1.4.9.

## Added

- Added OCR model selection for Files To Cards so OCR can use its own installed text model.
- Added per-feature AI context length settings for card autofill, grading, MCQ generation, Ask AI, Wikipedia breakdowns, Files To Cards, follow-ups, reinforcement cards, and stats summaries.
- Added Wikipedia breakdown support from Ask AI with cleaned article extracts and study-ready explanations.
- Added Ask AI image search and wiki/search action icons.

## Improved

- Improved AI model preflight handling so OCR model selection follows installed model availability.
- Improved Ask AI, autofill, grading, MCQ, and Files To Cards workers to use the configured context lengths.
- Prevented settings combo boxes, sliders, and spin boxes from changing accidentally while scrolling unless focused.

## Included

- Release artifacts for `1.5.9` use the `ONCard-Setup-1.5.9.exe` naming convention.
