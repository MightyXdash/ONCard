# ONCard v1.2.6

![ONCard 1.2.6 update banner](https://raw.githubusercontent.com/MightyXdash/ONCard/main/assets/updates/common/update_prompt_banner_16x9.png)

This release focuses on the Ask AI response experience and update readiness for the next minor version.

## Improved

- Ask AI now streams directly into the response overlay instead of being re-streamed a second time by the UI.
- The Ask AI response surface now sizes itself more intelligently to the content while staying within the approved maximum window size.
- Newly streamed Ask AI characters now fade in quickly for clearer incremental reading.
- Ask AI settings were simplified by removing the old local stream controls from the Settings dialog.

## Changed

- Removed the temporary expand button from the Ask AI overlay after switching to automatic height growth.
- Reset local release prep so `1.2.6` can be packaged cleanly with fresh installer/update assets.

## Notes

- Build and release artifacts for `1.2.6` should use the existing `ONCard-Setup-1.2.6.exe` naming convention.
- Version-specific update media can still fall back to the shared assets in `assets/updates/common` when `1.2.6` does not ship custom banners.
