# ONCard v1.1.6

This release focuses on interface polish, smoother motion, and cleaner interaction feedback.

## Improved

- Reworked search suggestions so they render above the card grid instead of pushing layout content downward.
- Improved search suggestion sizing and visibility so up to five results can appear cleanly without clipped rendering.
- Refined card-result entrance motion with smoother movement and stronger blur-based transition polish.
- Improved top-level tab transitions with a smoother animated slide path.
- Added selected-state white icon variants for the main `Create` and `Cards` navigation buttons.

## Changed

- Restricted `woosh` to navigation flows and switched most other interactions back to the standard click sound.
- Removed the launch sound effect when opening the app window.

## Notes

- Installer metadata and packaged update content were refreshed for version `1.1.6`.
- Existing packaged update banners continue to fall back safely to the shared common assets when version-specific images are not present.
