# ONCards v1.0.0-beta.2

Installer-based Windows beta release for ONCards.

![ONCards](https://github.com/user-attachments/assets/cd88364e-904a-41d1-87ff-0444d2cd70a7)

- Added a real Windows installer build with Nuitka + Inno Setup.
- ONCards now stores installed-app data outside the app folder, so updates do not wipe cards or settings.
- Added GitHub release update checks on startup.
- Update prompts can now show release notes and small images from the GitHub Release body.
- Kept the Ollama/model setup flow inside the app, so users install the app first and download models from ONCards itself.
