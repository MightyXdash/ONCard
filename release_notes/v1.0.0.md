# ONCard 1.0.0

The full ONCard 1.0 release is here.

- Added Files To Cards for turning slides, PDFs, and note images into queued study questions.
- Simplified setup so the app now uses one required Gemma model with a cleaner onboarding flow.
- Improved study mode with softer grading, an `I don't know` shortcut, easier follow-up help, and card options.
- Refined the Create flow, update experience, and overall UI polish across the app.
- Kept app updates separate from user study data so normal upgrades do not reset existing progress.
